# Flask CRUD API 

## Overview
This repo gives a sample CRUD API for Users functionality 

## Usage
 - Pre-requisite:

   `$ pip install flask`

## API's Supported
 - **L**ist: `GET /<entity>`
 - **C**reate: `POST /<entity>`
 - **R**etrieve: `GET /<entity>/<id>`
 - **U**pdate: `PUT /<entity>/<id>`
 - **D**elete: `DELETE /<entity>/<id>`
